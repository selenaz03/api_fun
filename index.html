Fun Fact



The API I selected picks a random fun fact and I display it on the screen.


The website that created this API is https://jsph.pl/ and I found it on the github page of the list of APIs that was listed in the AJAX readings module.


I used the AJAX GET method in order to retrieve the data in one way, and I used the Fetch fetch() function to retrieve the data the second way.


One application where this API would be helpful is possibly to display fun data in between screens when a site is loading. Another use for them could be to generate trivia questions or answers.
